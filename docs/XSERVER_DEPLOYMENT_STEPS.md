# XサーバーへのNext.jsアプリデプロイ手順（段階的ガイド）

## 📋 現在の状況

- ✅ Xサーバーのドメイン取得済み
- ⏳ デプロイ作業は未実施

## 🎯 目標

スマホでアクセスできるように、XサーバーにNext.jsアプリをデプロイする

## 📝 ステップ1: Xサーバーの環境確認

### 1-1. Xサーバーのコントロールパネルにログイン

1. Xサーバーのコントロールパネルにアクセス
2. ログイン情報を確認

### 1-2. Node.js機能の確認

Xサーバーのコントロールパネルで以下を確認：

- 「サーバー管理」→「Node.js設定」があるか確認
- Node.jsが利用可能か確認

**確認事項：**
- [ ] Node.js機能が利用可能か
- [ ] Node.jsのバージョン（18以上が推奨）

### 1-3. FTP/SSH情報の確認

- FTPサーバー名
- FTPユーザー名
- FTPパスワード
- SSH接続情報（利用可能な場合）

## 📝 ステップ2: プロジェクトの準備

### 2-1. 環境変数の設定ファイル作成

`.env.production` ファイルを作成して、本番環境用の設定を追加

### 2-2. ビルド設定の確認

`next.config.ts` をXサーバー用に調整

## 📝 ステップ3: ビルドとアップロード

### 3-1. ローカルでビルド

```bash
npm run build
```

### 3-2. ファイルのアップロード

必要なファイルをXサーバーにアップロード

## 📝 ステップ4: サーバーでの設定

### 4-1. Node.jsの起動

サーバー上でNext.jsアプリを起動

### 4-2. ドメインの設定

取得済みのドメインをアプリに紐付け

## 📝 ステップ5: 動作確認

### 5-1. スマホでのアクセステスト

実際にスマホからアクセスして動作確認

---

## 🔄 次のアクション

**まず、以下を教えてください：**

1. **XサーバーのコントロールパネルでNode.js機能が利用可能か**
   - 利用可能 → ステップ2に進む
   - 利用不可 → 別の方法を検討（静的エクスポート or Vercel）

2. **取得済みのドメイン名**
   - 例: `example.com` または `example.xsrv.jp`

3. **FTP/SSH情報**
   - FTPサーバー名
   - ユーザー名
   - パスワード（安全に管理）

この情報をいただければ、次のステップに進みます！
